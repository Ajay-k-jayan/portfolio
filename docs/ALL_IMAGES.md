## Image Sources

### CDN Providers

#### 1. jsDelivr CDN
- **Provider:** jsDelivr
- **Repository:** DevIcons (https://github.com/devicons/devicon)
- **Base URL:** `https://cdn.jsdelivr.net/gh/devicons/devicon/icons/`
- **Format:** SVG
- **Usage:** Primary source for technology icons

#### 2. Simple Icons CDN
- **Provider:** jsDelivr (Simple Icons)
- **Repository:** Simple Icons (https://github.com/simple-icons/simple-icons)
- **Base URL:** `https://cdn.simpleicons.org/`
- **Format:** SVG
- **Usage:** Brand icons and logos

### Image Loading Strategy

1. **Lazy Loading:** All skill images use lazy loading for performance
2. **Error Handling:** Fallback handling for failed image loads
3. **Caching:** CDN images are cached by the browser
4. **Format:** All icons are SVG for scalability and performance

### Image Mapping

The image URLs are mapped in `components/tabs/skills-tab.tsx` in the `getSkillImageUrl` function:

```typescript
const skillLogoMap: Record<string, string> = {
  'javascript': 'https://cdn.jsdelivr.net/gh/devicons/devicon/icons/javascript/javascript-original.svg',
  // ... more mappings
}
```

## Image Optimization Notes

1. **SVG Format:** All icons use SVG format for:
   - Scalability at any size
   - Small file sizes
   - Crisp rendering at all resolutions
   - Easy theming with CSS

2. **CDN Benefits:**
   - Fast global delivery
   - Automatic caching
   - No local storage needed
   - Always up-to-date icons

3. **Performance:**
   - Images load on-demand (lazy loading)
   - Error handling prevents broken images
   - Fallback mechanisms in place
